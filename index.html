UGENとは?

PCで作業中、つい娯楽サイトで遊んでしまう人向けの、ブラウザ監視＆サボりチクリツールです。

自己管理ができなくてサボってしまうという問題を解決したい
頑張りたいけど頑張れない人向けの
行動の強制を実現する
UGENです
これは人間関係を利用することができ
アラームによる行動管理とは違って、
他人を利用した強制機能が備わっています

使い方

このツールはGoogleChrome拡張機能を利用しています。

  • ストアを経由しない場合のインストール方法

    1. $ git clone git@github.com:enpitut/WakeUp.git
    2. GoogleChromeを開く
    3. 右上（GoogleChromeの設定）をクリック→「設定」をクリックして設定画面に移動する
    4. 「拡張機能」メニューを押して拡張機能画面に移動する
    5. 「デベロッパーモード」チェックボックスをチェックされた状態にする
    6. 「パッケージ化されていない拡張機能を読み込む」ボタンを押し、クローンしたリポジトリ内の「chrome_extension」フォルダを選択する
    7. ウィンドウ右上にUGENのアイコンが表示される

  • 事前設定

    1. UGENのアイコンをクリックする
    2. 「詳細設定」をクリックし、「Twitter連携」ボタンを押した後、Twitterの画面から連携アプリを認証する
    3. 表示されたPINコードを、ポップアップしたウインドウに入力する

  • 実行

    1. UGENを開き、タスクに集中したい時間を指定したあと、「スタート」ボタンを押すと監視が開始される
    2. タスクを完了した場合は「タスク完了」ボタンを押すと監視が終了する
